Bedrock Server

This is hosted in my office (Read Converted Garedn Shed)  and is an old Dell Inspirion 13 laptop repurposed with Windows 2022 Server and run Bedrock Dedicated Server ( Which you can download here ) 

The Spec of the machine has 16GB Ram and and older Intel I5. 

The Bedrock Server is pretty much a vanilla install with a couple of additions. The first is a quality of life installation of One Player Sleep from Foxy No Tail and Foxys Bedrock Server Software , with  some script for backups and map building (Unmined.net). 

Backup are done through the Windows Drive Snapshot feature that allows Windows to save change only data, thus keeping backups smaller and allow 15 mins snapshots to be taken. 

Mapping is an important part of any Minecraft server and Bedrock has suffered from a lack of tooling to automate this process. Unmined fixes this to some degree as it allows both Java and Bedrock Servers to either be viewed through a GUI or can produce a Web based map for deployment via a Web server. 

Once Unmined has created its map this uploaded to GitHub and then pushed to Cloudflares Static Website host which offers free Site hosting for static files. This process can be seen in more details on the GitHub Page.
